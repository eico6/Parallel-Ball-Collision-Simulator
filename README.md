# Parallel-Bouncing-Balls
- INF236 (Parallel Programming) course at UiB (Univeristy of Bergen), third compulsory assignment.
- by Eivind Hobrad Naasen & Daniel Ngyuen

# What is this?
- A parallel algorithm that effectively simulates a large amount of bouncing balls.
- Challange: resolve race conditions between ball collisions. 

# Development notes
- C++ version: g++.exe (MinGW-W64 x86_64-ucrt-posix-seh, built by Brecht Sanders, r8) 13.2.0
- Graphics library: raylib version 5.5
